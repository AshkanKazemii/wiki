## راه اندازی اولیه

- نصب داکر
- دانلود پروژه از گیت هاب 
- اجرای دستور زیر برای build  گرفتن از پروژه
```
docker-compose up -d --build
```
-  در مرورگر فقط localhost:8080  را بزنید پروژه اجرا میشود
